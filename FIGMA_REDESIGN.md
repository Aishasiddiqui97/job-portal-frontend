## 🎨 **Design Specifications:**

### **Colors:**
```css
Primary Blue: #2563eb
Accent Red: #ef4444
Gray Background: #f9fafb
White: #ffffff
Text: #111827
Border: #e5e7eb
```

### **Spacing:**
```css
Section Padding: py-10 (40px)
Container Max Width: max-w-7xl (1280px)
Grid Gaps: gap-4 (16px)
Card Padding: p-4 (16px)
```

### **Typography:**
```css
Font Family: Inter
Heading: text-2xl font-bold (24px bold)
Body: text-base (16px)
Small: text-sm (14px)
Tiny: text-xs (12px)
```

### **Border Radius:**
```css
Cards: rounded-lg (8px)
Buttons: rounded-md (6px)
Icons: rounded / rounded-full
```
